
# Roadmaps

This is the method I use for learning and remember **binaries** and **networks** while I'm solving challenges.

## Usage

### Clone the repository

`git clone https://github.com/PFPE20/roadmaps.git`

### Create a syslink

`ln -s /path/to/knowledge.sh /path/to/directory`

### Add to PATH:

In your `.*rc` file (zshrc, bashrc, etc...) add this line:

`export $PATH="$PATH:/path/to/directory"`

**OR**

Create an alias in the same file:

`alias roadmap=/path/to/directory/knowledge.sh`
